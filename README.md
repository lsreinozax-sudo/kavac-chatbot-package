# Kavac Chatbot para Laravel

Este paquete integra el chatbot Kavac (API Django) en cualquier proyecto Laravel.

## Requisitos

- PHP ^8.1
- Laravel ^9.0|^10.0|^11.0|^12.0
- API de Kavac corriendo (ver documentación aparte)

## Instalación

```bash
composer require tu-vendor/kavac-chatbot
